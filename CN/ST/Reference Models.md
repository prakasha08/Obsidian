#### 1. OSI (Open Systems Interconnection) Model

- Developed by ISO.
    
- Contains **7 layers**:
    
    1. Physical
        
    2. Data Link
        
    3. Network
        
    4. Transport
        
    5. Session
        
    6. Presentation
        
    7. Application
        

Each layer has specific tasks, and layers interact only with their direct neighbors.

#### 2. TCP/IP Model
![[Pasted image 20250424224616.png]]
- Used practically across the internet.
    
- Contains **4 layers**:
    
    1. Network Access
        
    2. Internet
        
    3. Transport
        
    4. Application
